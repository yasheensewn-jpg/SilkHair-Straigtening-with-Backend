# Silky Hair Straightening App

A sleek and modern booking application for a hair salon, featuring separate interfaces for customers to book appointments and for the owner to manage and cancel bookings.

## Features

- **User Interface**: Book appointments, view history, manage profile.
- **Owner Interface**: Dashboard to manage bookings, set availability, and view client details.
- **Real-time availability**: Smart slot calculation based on service duration.
- **Authentication**: Role-based access (Client vs Owner).

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Locally**
    ```bash
    npm run dev
    ```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS (via CDN)
